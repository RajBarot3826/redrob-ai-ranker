# RedRob AI Ranker — Intelligent Candidate Discovery

Built for the **India Runs Data & AI Challenge** by Redrob AI × Hack2Skill.

## How to use
1. Upload the `sample_candidates.json` from the hackathon bundle
2. The system ranks candidates for the Senior AI Engineer role
3. View scores, reasoning, and download the submission CSV

## Architecture
3-stage hybrid pipeline: Coarse Filter → Semantic Ranking → Fine Re-Ranking
